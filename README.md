# README

### Ruby version 3.0.0

# Configuration
To install project dependencies, run: 
```
bundle install
```
Remember to edit database.yml file with your database information

# Database creation
To create the database, run:
```
rails db:create
```
# Database initialization
To initialize the database, run:
```
rails db:migrate
```
